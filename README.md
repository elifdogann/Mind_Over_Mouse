# Mind Over Mouse - VR Parkour Project

This project is a  VR parkour experience developed for the **IARVR course at TU Darmstadt during Winter Semester 2025/26**.

## Overview

The game combines **gesture-based locomotion**, **environmental traversal**, and **tool-mediated object interaction** in a virtual environment. Instead of relying only on conventional movement and interaction patterns, the project explores alternative control mechanics and gameplay flow.

Players move through the environment using controller-based gestures, collect thematic objects, and complete a small interaction puzzle by using a knife to manipulate a key and unlock a chest.

## Main Features

- Gesture-based locomotion
- Controller-driven avatar movement
- Thematic collectible objects
- Tool-based object interaction
- Snap-assisted key placement
- Chest unlock and book reveal sequence
